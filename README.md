# Investment Portfolio Optimization

## Overview

This project focuses on optimizing an investment portfolio using quantitative finance techniques. The objective is to maximize returns while minimizing risk through effective asset allocation.

## Features

* Portfolio return and risk calculation
* Efficient frontier visualization
* Sharpe ratio optimization
* Diversification strategies

## Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* SciPy

## Methodology

1. Data collection and preprocessing
2. Return and covariance calculation
3. Portfolio simulation
4. Optimization (Sharpe ratio / minimum volatility)

## Results

The model identifies optimal portfolios that improve risk-adjusted returns and demonstrate effective diversification.

## How to Run

```bash
git clone https://github.com/NKS-way/Investment-Portfolio-Optimization.git
pip install -r requirements.txt
python main.py
```
